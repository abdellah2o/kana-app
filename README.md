# KANA APP

The website to learn the Japanese kana (hiragana or katakana).
---

## Stack

Front : React (Vite)
Back : soon...
DB : soon...


## Features

- Study tab with a card for every kana and its romanization
- Quiz tab to test your memorization
- The quiz have a score system which lets you know what is your best score and try to beat it
